## 🎉 亮点功能

### 🧭 Security List 菜单式管理

全新的 `/sl_menu` 命令提供交互式向导，让 Security List 规则管理变得简单直观：

```
📝 查看安全列表

⬇️ 新增入站    ⬆️ 新增出站
🚫 删除入站    ❌ 删除出站
🔁 替换入站    ♻️ 替换出站
```

**核心特性**：
- 🎯 **实例优先**：从实例视角管理 Security List，无需记忆 OCID
- 🧙 **智能向导**：协议 → CIDR → 端口 → 描述 → 确认，一步步引导完成
- ⬅️ **可回退**：支持返回上一步修改，不怕填错
- 🔒 **严格校验**：CIDR 和端口输入自动验证，防止配置错误
- 🇨🇳 **中文友好**：按钮、提示、错误信息全中文

## ✨ 新增功能

### 智能向导流程

#### 1. 协议选择（中文化）
- **TCP (传输控制)** - 传输控制协议
- **UDP (用户数据报)** - 用户数据报协议
- **ICMP (网络控制)** - 互联网控制消息协议
- **ALL (所有协议)** - 放行所有流量

#### 2. CIDR 输入
**快捷按钮**：
- `0.0.0.0/0` - 允许所有 IPv4
- `::/0` - 允许所有 IPv6
- `10.0.0.0/8` - 私有网段 A 类
- `192.168.0.0/16` - 私有网段 C 类
- ✍️ 自定义输入

**自定义输入支持**：
- ✅ IPv4 CIDR：`192.168.1.0/24`
- ✅ IPv6 CIDR：`2001:db8::/32`
- ❌ 非法格式自动拒绝并提示

#### 3. 端口配置
**快捷按钮**：
- `22` - SSH
- `80` - HTTP
- `443` - HTTPS
- `25000-25100` - 端口范围示例
- ✍️ 自定义输入

**自定义输入支持**：
- 单端口：`8080`
- 端口范围：`3000-4000`
- 验证范围：1-65535
- 逻辑检查：起始 ≤ 结束

**智能跳过**：
- ALL、ICMP、ICMPv6 协议自动跳过端口步骤

#### 4. 描述填写
- 可选字段
- 支持中文描述
- 发送 `-` 表示留空

#### 5. 确认提交
- 预览所有参数
- 显示完整配置
- 支持返回修改
- 确认后执行

## 🎨 用户体验

### 图标区分度优化
- ⬇️ **入站** - 粗箭头向下（流量进入）
- ⬆️ **出站** - 粗箭头向上（流量离开）
- 🚫 **删除入站** - 禁止标志
- ❌ **删除出站** - 叉号
- 🔁 **替换入站** - 顺时针循环
- ♻️ **替换出站** - 回收标志

### 按钮中文化
- **修改前**：`Default Security List fo` (截断)
- **修改后**：`默认安全列表 vcn-xxx`

### 返回链路
- 确认页 → 描述页
- 描述页 → 端口页（或 CIDR 页）
- 端口页 → CIDR 页
- CIDR 页 → 协议页
